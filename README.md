# Stock Market Prediction

This project predicts stock market trends using machine learning models.
It has a Flask-based web interface where users can input stock names and view predicted trends.

## Features
- Machine Learning Model (Linear Regression / LSTM)
- Flask web interface
- Data visualization with Matplotlib

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run app: `python app.py`
3. Open `http://localhost:5000` in browser.
