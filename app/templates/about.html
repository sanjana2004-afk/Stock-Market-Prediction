About This Project

This project is a stock trend prediction web application built using Flask, Keras (for the LSTM model), and Yahoo Finance data (via `yfinance`).

  • It fetches historical stock data
  • Calculates EMAs (Exponential Moving Averages)
  • Uses a deep learning model to forecast future prices
  • Visualizes predictions vs actual prices

You can input any stock ticker (e.g., AAPL, MSFT, TATASTEEL.NS) and view its trend predictions and charts.
